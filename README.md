# BUAA_2026_OS
这里提供了我上OS课程的每个lab的实验报告与代码，虽然课程组也发过开源的代码，但是我个人觉得课程组的代码存在一些小bug，但是可能是因为数据点太弱导致没测出来，且课程组开源的代码有一些地方确实过于精简，导致理解困难，而我的代码应该就会好理解一些。

OS课下代码也没有抄袭一说，如果你觉得我的代码对你有帮助，请点个收藏。

我们OS课程是基于微内核的思想进行设计的，即把基础的功能留在内核中，而把其他大部分功能都留在用户程序中，这样不仅利于OS的移植，同时某个环节的崩溃（如文件系统的崩溃）也不会导致内核的崩溃，这是一个极好的设计。

但是这个微内核还是有些不足的，像是它竟然把部分异常处理放在用户空间去执行，说实话这个不仅麻烦，而且用处不大,并且可能部分程序还存在代码冗余和一些评测测不出来的小bug等。但是瑕不掩瑜，总体上来说，个人觉得这还是一个比较成功的系统，值得大家多花时间去品味其妙。
