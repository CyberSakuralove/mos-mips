# MOS – MIPS 微内核操作系统

本项目是 **操作系统课程** 的完整实验代码与实验报告。  
实现了基于 **MIPS 4Kc** 架构、采用 **微内核设计** 的教学操作系统 **MOS**，包含内核启动、内存管理、进程调度、系统调用、写时复制（COW）、进程间通信（IPC）、简单文件系统等核心模块。

> 课程组提供了基础框架，**所有核心功能的实现及绝大部分代码均独立完成**。相比课程组提供的参考实现，本项目的代码结构更清晰、注释更详尽，修复了若干评测未覆盖的边界 Bug，易于理解与二次开发。

## 1. 项目亮点

- **微内核架构**：文件系统、设备驱动运行在用户态，内核仅提供 IPC 和页映射等基础服务，崩溃隔离性强。
- **完整的内存管理**：物理内存（链表法）+ 虚拟内存（两级页表 + TLB 重填 + 页目录自映射）。
- **进程与异常**：进程控制块（Env）、时钟中断驱动的时间片轮转调度、TLB 缺失/页写入异常处理。
- **系统调用与 COW**：实现了 `fork` 系统调用，并通过 **写时复制（Copy‑on‑Write）** 优化内存使用，用户态页写入异常处理。
- **进程间通信（IPC）**：基于共享页面的消息传递机制，支持文件系统服务调用。
- **简单文件系统**：磁盘块缓存、目录/文件结构（直接+间接指针）、IDE 驱动（用户态通过系统调用访问 MMIO）。
- **代码质量**：消除轮询，使用链表宏（`LIST_ENTRY`, `TAILQ`）管理空闲资源，关键函数均有详细注释。

## 3. 技术栈

- **语言**：C（主）、MIPS 汇编、Makefile、Shell
- **工具链**：mips‑linux‑gnu‑gcc / ld, QEMU (MIPS malta), GDB
- **构建**：GNU Make（支持增量编译、模块化子目录）
- **版本控制**：Git（实验发布与提交均基于 Git）

## 4. 如何运行（简要）

> 详细环境配置参见课程指导书，此处仅提供核心命令。

```bash
# 克隆本仓库
git clone https://github.com/yourname/mos-mips.git
cd mos-mips

# 编译内核（默认包含全部实验模块）
make

# 启动 QEMU 模拟器运行内核
make run

# 运行指定单元测试（如 Lab2 物理内存管理测试）
make test lab=2_1 && make run

5. 项目结构
text
mos-mips/
├── init/          # 内核入口 (start.S, init.c)
├── kern/          # 内核核心代码（内存管理、进程、异常、系统调用）
├── lib/           # 内核与用户态共用库（printf, string, elf 加载等）
├── include/       # 全局头文件（mmu.h, env.h, trap.h, queue.h 等）
├── user/          # 用户程序（测试用例：fktest, pingpong, fstest...）
├── fs/            # 文件系统服务进程（用户态）
├── tools/         # 构建辅助工具（readelf, fsformat）
└── Makefile       # 顶层构建脚本
